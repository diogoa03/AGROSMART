Bem-vindo à documentação do AgroSmart!
=====================================

Esta é a documentação oficial do sistema AgroSmart, uma solução de gestão agrícola inteligente.

Características Principais
------------------------
* Monitoramento em tempo real de dados meteorológicos
* Sistema de recomendações automatizado
* Interface Web responsiva
* Integração com OpenWeatherMap API
