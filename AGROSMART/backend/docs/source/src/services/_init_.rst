Services Package
================

This package contains the services used in the AGROSMART project. The services are responsible for handling various functionalities related to weather recommendations, notifications, and other core operations.

Available Services
------------------

- **NotificationService**: Manages notifications based on weather conditions and recommendations.
- **RecomendacaoService**: Provides recommendations for grape cultivation based on weather data.
- **WeatherService**: Fetches current weather data and maintains a history of weather conditions.
